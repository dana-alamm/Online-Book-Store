# Online Book Store App 📚

A modern mobile application developed using **Flutter** and **Firebase**, designed to provide a seamless experience for browsing and managing a digital book library.

## ✨ Key Features
* **Firebase Authentication:** Secure user sign-up and login for personalized library access.
* **Book Catalog:** Dynamic list of books fetched in real-time from **Cloud Firestore**.
* **Categories & Search:** Filter books by genre, author, or title.
* **Image Hosting:** High-quality book covers managed through **Firebase Storage**.
* **User Profile:** Manage personal information and saved/purchased books.

## 🛠️ Tech Stack
* **Frontend:** Flutter & Dart.
* **Backend:** Firebase (Auth, Firestore, Storage).
* **IDE:** Android Studio / VS Code.

## 🚀 Setup & Installation
1. Clone the repository.
2. Run `flutter pub get`.
3. **Note:** To connect with Firebase, place your `google-services.json` (Android) or `GoogleService-Info.plist` (iOS) in the appropriate directories.

This project demonstrates my ability to integrate **Cloud Infrastructure (Firebase)** with **Mobile UI (Flutter)** to create a functional E-commerce/Utility application.
